# Scream Race KZ

Голосовая браузерная гонка для мероприятий. Ведущий открывает комнату на большом экране, игроки подключаются со смартфонов и двигают машинки громкостью голоса.

## Что внутри

- `index.html` — главная страница
- `host.html` — экран ведущего
- `player.html` — экран игрока
- `style.css` — стили
- `host.js` — логика комнаты и гонки
- `player.js` — микрофон и подключение игрока

## Быстрый запуск

Можно открыть `index.html` двойным щелчком для просмотра интерфейса. Для полноценного соединения и микрофона лучше использовать локальный сервер:

```bash
python -m http.server 8000
```

Затем откройте:

```text
http://localhost:8000
```

## Публикация на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы из этой папки в корень репозитория.
3. Откройте `Settings` → `Pages`.
4. В разделе `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/root`.
6. Откройте выданную GitHub Pages ссылку.

## Как играть

1. На компьютере/телевизоре откройте `host.html`.
2. Игроки сканируют QR-код или открывают `player.html` и вводят код.
3. Игроки включают микрофон.
4. Ведущий нажимает «Старт гонки».
5. Громкость голоса двигает машинки.

## Важно

- Проект не требует npm, React, Vite или сборки.
- Для связи между устройствами используется PeerJS через CDN и публичный PeerJS-сервер.
- Для QR-кода используется библиотека QRCode через CDN.
- Нужен интернет.
- Микрофон в браузере работает на HTTPS или `localhost`.
- Ведущий должен держать страницу открытой во время игры.
- Публичный PeerJS-сервер подходит для тестов и небольших игр, но для коммерческого мероприятия лучше развернуть собственный PeerServer.
