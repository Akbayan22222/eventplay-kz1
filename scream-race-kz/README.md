# Scream Race KZ

Готовая браузерная игра без React/Vite/npm.

## Что внутри

- `index.html` — стартовая страница
- `host.html` — экран ведущего
- `player.html` — страница игрока
- `style.css`
- `host.js`
- `player.js`
- `assets/` — фото машин

## Как работает

1. Ведущий открывает `host.html`.
2. На экране появляется код комнаты и QR-код.
3. Игрок сканирует QR-код, вводит имя и разрешает микрофон.
4. После старта ведущего машина едет от громкости голоса.

## Загрузка на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи все файлы из этой папки в корень репозитория.
3. В `Settings → Pages` выбери ветку `main` и папку `/root`.
4. Открой ссылку GitHub Pages.

## Важно

- Для работы микрофона нужен `https` или `localhost`.
- На GitHub Pages микрофон работает нормально.
- Для соединения используется PeerJS Cloud.
